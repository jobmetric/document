# Laravel Flow

Workflow for Laravel - Create different workflows in your Laravel projects.

## Overview

`Workflow` streamlines the creation of processes within your program. By entrusting process creation to system administrators, you alleviate yourself from redundant tasks. This methodology not only allows you to tackle smaller tasks more efficiently but also empowers you to craft comprehensive projects with ease.

## Installation

```bash
composer require jobmetric/laravel-flow
```
