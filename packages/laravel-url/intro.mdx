# Create SEO-Friendly URLs. Manage Slugs Effortlessly.

**Laravel URL** revolutionizes how you manage URLs and slugs in Laravel applications. Stop wrestling with manual URL generation and start building SEO-friendly, versioned URLs that automatically handle redirects, conflicts, and cascading updates. This is where powerful URL management meets developer-friendly simplicity‚Äîgiving you complete control over your application's routing without the complexity.

## Why Laravel URL?

### Automatic URL Versioning

Laravel URL automatically tracks the complete history of your URLs. When a slug or parent path changes, the package creates a new version while preserving the old one for SEO-friendly 301 redirects. Your URLs evolve with your content, but old links never break.

### Intelligent Conflict Detection

Never worry about duplicate URLs again. Laravel URL enforces global uniqueness for active URLs, automatically detecting conflicts and preventing data integrity issues. The package handles edge cases like soft deletes, restores, and cascading updates with built-in conflict resolution.

### Cascading URL Updates

When a parent model's slug changes (like a category), Laravel URL automatically updates all child URLs (like products). This cascading behavior ensures your URL structure stays consistent without manual intervention.

### Smart Fallback Routing

Laravel URL provides a powerful fallback route system that resolves any unmatched path using your versioned URL database. Active URLs trigger events for custom routing, while legacy URLs automatically redirect (301) to their canonical versions‚Äîperfect for SEO.

## What is URL Management?

URL management is the foundation of SEO-friendly web applications. Consider an e-commerce product that lives at `/shop/laptops/macbook-pro-14`. When the category changes from "laptops" to "computers", the product URL should automatically update to `/shop/computers/macbook-pro-14`, while the old URL redirects to the new one.

Each URL consists of:
- **Slug**: A friendly, URL-safe identifier (e.g., `macbook-pro-14`)
- **Full URL**: The complete path including parent segments (e.g., `/shop/laptops/macbook-pro-14`)
- **Versioning**: Historical tracking for redirects and SEO
- **Collections**: Optional grouping for organizing slugs by context

### Power at Your Fingertips

With Laravel URL, you have complete control over your URL structure. Define hierarchical URLs, implement automatic versioning, handle conflicts gracefully, support soft deletes and restores, and generate SEO-friendly redirects‚Äîall without writing complex routing logic.
