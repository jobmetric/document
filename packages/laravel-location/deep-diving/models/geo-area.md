# GeoArea Model

Represents a reusable grouping of multiple locations.

Geo areas typically use `HasLocation` in **multiple** mode.

## Namespace

```php
JobMetric\Location\Models\GeoArea
```

## Table

- `location.tables.geo_area` (default: `location_geo_areas`)

## Relationships

- `geoAreaRelations()`
- `locations()` (via `HasLocation`)
